# Replace Meiryo

This font is combination fonts with [Roboto](https://github.com/googlefonts/roboto/releases/download/v2.138/roboto-android.zip) and [Source Han Sans CN](https://github.com/adobe-fonts/source-han-sans)

This project is for english users of Windows, to fix the ugly CJK characters.

On mostly Asia website **Meiryo** and **MS Gothic** is typically listed as the fallback font. But on non-CJK system, **Meiryo** isn't installed by default and **MS Gothic** is very very ugly.

So, this project provide a fake **Meiryo**. Installing this can help with the display of CJK characters in Spotify Desktop.

## Usage

If you aren't installed **Meiryo** font, just install it and enjoy.
